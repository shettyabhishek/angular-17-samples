@defer block & @placeholder
@defer{
A defer block ensures that the block is loaded a separate chunk of js file on demad. Thus, enhancing performance by reducing the rendering time
  • @placeholder only works with @defer
  • @placeholder is not a loading indicator block
  • @placeholder appears only when in the background the @defer section is preparing for rendering
  • @placeholder can be made to display for a specific time interval using minimum attribute. E.g.-> @placeholder(minimum 3s)
  • @placeholder is eagerly loaded and is not moved into a chunk file.
} @placeholder (minimum 5s){
The place holder gets shown for a specific interval of time.
}
@defer block & @loading
@defer{
@loading section appears while the @defer block is about to load. The behaviour looks similar to @placeholder, but both of these serve different purpose
  • @loading only works with @defer
  • @loading can also be made to display for a specific time interval using minimum attribute. E.g.-> @loading(minimum 3s)
  • @loading can be made to display only if the defer block is taking more than specified time using an after attribute E.g.-> @loading(minimum 3s; after 1s)
@error block is displayed in case the @defer block fails to load for some reason.
} @loading (minimum 3s){
The content is about to load. Loading ...
}
