@for - template annotation
The @for block repeatedly renders content of a block for each item in a collection.The value of the track expression determines a key used to associate array items with the views in the DOM. This also supports $index, $first, $last, $count, $even and $odd.
When the items are available in the list

    • }
When the list is empty @empty block can be used to show a default block
    @for(itm of noItemsArray;track $index){
  • {{$index+1}}. {{itm}}
    • } @empty {
